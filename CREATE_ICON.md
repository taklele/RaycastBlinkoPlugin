# 如何创建图标

## 方法 1: 使用 SF Symbols (macOS 推荐)

1. 打开 SF Symbols 应用
2. 搜索 "book" 或 "note.text"
3. 导出为 PNG，512x512
4. 重命名为 icon.png，放到 assets/ 目录

## 方法 2: 使用在线工具

访问以下网站之一：
- https://www.flaticon.com/ (搜索 "journal" 或 "diary")
- https://icons8.com/ (搜索 "notebook")

下载 512x512 的 PNG 图标。

## 方法 3: 使用 Emoji 转图标

1. 访问 https://emoji-to-png.vercel.app/
2. 选择一个 emoji（如 📔 📝 ✍️）
3. 下载为 PNG
4. 放到 assets/icon.png

## 临时方案

如果只是测试，可以使用任意 PNG 图片，只要是 512x512 即可。
